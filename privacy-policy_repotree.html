Privacy Policy

Last Updated: November 7, 2025

Data Collection

RepoTree does not collect, store, or transmit any personal data or user information.

Data Usage

This extension:

  • Only operates on GitHub.com pages when you explicitly click the Tree button
  • Makes API requests directly to GitHub's public API on your behalf
  • Temporarily caches repository tree data in your browser's memory (cleared on page reload)
  • Does not send any data to third-party servers
  • Does not track your browsing activity

Permissions

The extension requires the following permissions:

  • host_permissions (github.com): To inject the Tree button and interact with GitHub pages

Third-Party Services

This extension makes requests to GitHub's public API (api.github.com) to retrieve repository file trees. These requests are subject to GitHub's own privacy policy and terms of service.

Changes to This Policy

Any changes to this privacy policy will be reflected in the extension's repository and updated version.
